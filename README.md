# CyberSecurity-Tools
CyberSec tools built by me for various functionalities

Network-PortScanner:
A Simple PortScanner, built using python, using simple if else and try loops. To use it, run the program in a terminal with the ip of your router/network device and it will scan all the open ports in the range from 50-85. Can be further modified to serve you own needs.

1.4 Billion Text Credentials Analysis Parser:
A bash script/program to parse through Credentials leaked in the https://github.com/philipperemy/tensorflow-1.4-billion-password-analysis repo. Download the Credentials data and place the script inside it.
